About this app

Antisocial Feed is a read-later app with a quiet superpower: it remembers the shape of everything you've saved and, when you're looking for something to read next, suggests from your own library — never from a feed, a partner catalog, or anyone else's reading list.

The server is an optional component, it is small open-source component you can run yourself. Articles are embedded into small vectors so the app can find related material in milliseconds, even offline. Those vectors never leave your box.

What you get

- One-tap save from Safari, Messages, or any share sheet
- Links open in an in-app webview — the live page, as written
- Private recommendations that respect your taste
- Highlights and notes that export as Markdown
- Full offline mode — library cached, saves queued
